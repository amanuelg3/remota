Remota is an application which can be used as an air mouse and an air keyboard. You can remote control your host PC with simple user interfaces like a touch pad or Nintendo Wii motion plus.

For first use

For first use, you need some preparations. Please go to the Help page (Press the menu key and select "Help".

Server application update

Because wrong serial port names are shown on the Windows7 + .NET Framework 3.5, we update the server application to v0.9.0.1. v0.9.0.1 needs .NET Framework 4.0.
