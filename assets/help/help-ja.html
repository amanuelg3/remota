はじめに

Remotaはエアマウスやエアキーボードとして使用できるアプリケーションです。 タッチパッドや任天堂のWii motion plusのようなシンプルなユーザインタフェースで、ホストPCを遠隔操作できます。

必要なもの

  • ホストPC上のBluetoothドングル
  • ホストPC上のサーバアプリケーション
RemotaはBluetooth Serial Port Profile(SPP)上で動作します。そのため、Bluetoothドングルが必要です。 携帯機器からホストPCへのデータは、Bluetoothシリアルポート上を一般的なデータとして流れます。 このデータを解析するためのサーバアプリケーションが必要です。

準備

ペアリング

まずはじめに、携帯機器とホストPCを次の手順でペアリングする必要があります。

Windows bluetoothスタックの場合:

  1. メニューキーを押下し、"発見可能にする"を選択する
  2. ホストPCから携帯機器をサーチする
    1. タスクトレイ上のBluetoothアイコンを押下する
    2. "デバイスを追加"を押下する
    3. 携帯機器を選択する
  3. ペアリングのパスキーを入力する

Toshiba bluetoothスタックの場合:

  1. ホストPC: ホストPCを発見可能にする
    1. スタート -> すべてのプログラム -> TOSHIBA -> Bluetooth -> Bluetooth設定 -> Bluetooth -> オプション -> セキュリティ -> レベルのカスタマイズ
    2. モードのすべてのチェックを外す
  2. Remotaを起動し、 メニューキーを押下して、"デバイスと接続"を選択する
  3. "デバイスをスキャン"を押下する
  4. "新しいデバイス"にあるホストPCを選択する
  5. ペアリングのパスキーを入力する

サーバアプリケーションのインストール

使用方法

着信COMポートを探す

ホストPC上のサーバアプリケーションは、Remotaからのデータを受けとるために着信COMポートを監視しなければなりません。 着信COMポートは次の手順で探せます。

Windows bluetoothスタックの場合:

  1. タスクトレイ上のBluetoothアイコンを押下する
  2. "設定を開く"を押下する
  3. "COMポート"タブを押下する
  4. ポート名が携帯機器で、向きが着信になっているCOMポートを探す

Toshiba bluetoothスタックの場合:

  1. コントロールパネルを開く
  2. "Bluetooth Local COM"を押下する
  3. 所有者が"LocalCOM-Server[SerialPort(LocalCOM)]"となっているCOMポートを探す

接続

次の手順で携帯機器をホストPCに接続してください。 もし携帯機器にジャイロセンサが搭載されている場合、 モーションパッドモードを使用できます（接続する前に、 メニューキーを押下して"設定"を選択し、"モード"で"モーションパッドモード"を選択してください）。
  1. ホストPC: サーバアプリケーションを起動する
  2. ホストPC: 着信COMポートを選択する：着信COMポートはこの手順で確認できます
  3. ホストPC: スタートボタンを押下する
  4. メニューキーを押下し、"デバイスと接続"を選択して、"ペア設定済み"にあるホストPCを選択する
接続が確立されると、選択したモードに応じた画面が表示されます。

タッチパッドモード

タッチパッド画面には、次の五つの領域があります：左ボタン、右ボタン、キーボードボタン、スクロール領域とタッチパッド領域

タッチパッド画面

移動
タッチパッド領域で指を移動させる
左ボタンクリック
タッチパッド領域をタップする、または、左ボタンをタップする
右ボタンクリック
右ボタンをタップする
ドラッグアンドドロップ
左ボタンを指で押さえ、別の指をタッチパッド領域で移動させる
スクロール
スクロール領域で指を移動させる
キー入力
キーボードボタンをタッチすると次のようなキーボード画面が表示される

キーボード画面

モーションパッドモード
ジャイロセンサが必要です。
画面をタッチすると、次の四つの領域がタッチした位置のまわりに表示されます：左ボタン、右ボタン、キーボードボタンとスクロールボタン

画面をタッチすると表示されるボタン

移動

右ボタンクリック
