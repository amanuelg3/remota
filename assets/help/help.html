Introduction

Remota is an application which can be used as an air mouse and an air keyboard. You can remote control your host PC with simple user interfaces like a touch pad or Nintendo Wii motion plus.

Requirements

  • A Bluetooth dongle on your host PC
  • A server application on your host PC
Remota runs on the Bluetooth Serial Port Profile(SPP), so you need a Bluetooth dongle. Packets from your mobile device to your host PC go on a Bluetooth serial port as general data, so your host PC needs a server application for analyze this data.

Preparations

Pairing

First, you need pairing your mobile device to your host PC with the following steps.

Case the Windows bluetooth stack:

  1. Press the menu key and select "Make discoverable"
  2. Search your mobile device from your host PC
    1. Press the Bluetooth icon on the tasktray
    2. Press "Add a new Bluetooth Device"
    3. Select your mobile device
  3. Enter pairing passkey

Case the Toshiba bluetooth stack:

  1. Host PC: Make your host PC discoverble
    1. Start -> All Programs -> TOSHIBA -> Bluetooth -> Bluetooth Settings -> Bluetooth -> Options -> Security -> Custom Level
    2. Check off all options in the "Mode" section
  2. Run Remota, press the menu key and press "Connect a device"
  3. Press "Scan for devices"
  4. Select your host PC in the "New devices" section
  5. Enter pairing passkey

Install server application

Next, install the server application(RemotaHost*****.zip) from the following URL to your host PC.

http://code.google.com/p/remota/downloads/list

How to use

Find the incoming COM port

The server application on your host PC must listen the incoming COM port to receive packets from Remota. The incoming COM port number can be found by the following steps.

Case the Windows bluetooth stack:

  1. Press the Bluetooth icon on the tasktray
  2. Press "Open Bluetooth Settings"
  3. Press the "COM Ports" tab
  4. Find the COM port whose name is your mobile device and direction is incoming

Case the Toshiba bluetooth stack:

  1. Open the control panel
  2. Press "Bluetooth Local COM"
  3. Find the COM port whose owner is "LocalCOM-Server[SerialPort(LocalCOM)]"

Connect

Connect your mobile device to your host PC in the following steps. If your mobile device has a gyroscope, you can use the motion pad mode(before connect, press the menu key and select "Settings", tap "Mode" and select "Motion pad mode").
  1. Host PC: Run the server application
  2. Host PC: Select the incoming COM port which can be found by this steps
  3. Host PC: Press the start button
  4. Press the menu key, select "Connect a device" and select your host PC in the "Paired devices" section
When the connection is complete, the screen corresponding the selected mode will appear.

Touch pad mode
