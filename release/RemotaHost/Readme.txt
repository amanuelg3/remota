----------------------------------------
RemotaHost v0.9.0.1
----------------------------------------

This is a host application for the Android application "Remota".
Remota sends packets which indicate mouse or keyboard event.
This application analyze it and perform events on the Windows PC.

Requirements:
Microsoft .NET Framework 4.0

Download site:
http://www.microsoft.com/downloads/en/details.aspx?FamilyID=9cfb2d51-5ff4-4491-b0e5-b386f32c0992&displaylang=en

How to install:
unzip this folder

How to uninstall:
delete this folder
