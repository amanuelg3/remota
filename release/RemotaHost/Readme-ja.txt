----------------------------------------
RemotaHost v0.9.0.1
---------------------------------------

このアプリはアンドロイドアプリ"Remota"のホスト側アプリケーションです。
Remotaはマウスやキーボードイベントデータを送信します。
このアプリはそのデータを解析し、Windows PC上でイベント発行します。

必要なもの：
Microsoft .NET Framework 4.0が必要です

Download site:
http://www.microsoft.com/downloads/details.aspx?displaylang=ja&FamilyID=9cfb2d51-5ff4-4491-b0e5-b386f32c0992

インストール方法：
このフォルダを解凍する

アンインストール方法：
このフォルダを削除する
