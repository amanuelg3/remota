----------------------------------------
RemotaHost v0.9.0.0
---------------------------------------

このアプリはアンドロイドアプリ"Remota"のホスト側アプリケーションです。
Remotaはマウスやキーボードイベントデータを送信します。
このアプリはそのデータを解析し、Windows PC上でイベント発行します。

このアプリの動作には
  .NET framework 3.5
が必要です。事前にインストールをお願いします。

インストール方法：
このフォルダを解凍する

アンインストール方法：
このフォルダを削除する
